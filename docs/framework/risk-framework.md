## Sharpe Ratio Optimization

We optimize strategies to maximize the **Sharpe Ratio**, ensuring returns justify the volatility.

### Formula

$$Sharpe = \frac{R_p - R_f}{\sigma_p}$$

Where:
- $R_p$ = Portfolio return
- $R_f$ = Risk-free rate
- $\sigma_p$ = Portfolio volatility (standard deviation)

Unlike Sharpe, Sortino only penalizes **downside volatility**, making it more relevant for asymmetric strategies.

$$Sortino = \frac{R_p - R_f}{\sigma_d}$$
